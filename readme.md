## Фронт

* nodejs version v16.14.0
* npm version 8.3.1
* npx version 8.3.1

В папке `front/src/server` указать ссылку на бек, после чего запустить

## Бек

Сделать стандартные настройки в файле .env
В php artisan сделать миграции и сиды
